Methodology · 6-factor scientific model

NIRF rank alone does not predict whether a college will fit your child.

Real choice-filling optimises for branch, placement, location, peer quality, closing-rank trends — not just a single ranking column. EduAakashaa's matrix uses a six-factor composite model: each input is normalised to 0–1, weighted, and combined into one score. A small bias-correction layer (α) rebalances for "hidden gems" the rankings undervalue.

F·01

People Choice (PC)

PC = 1 − (closing ÷ maxClosing) with trend correction ×(1 + Δ). Lower closing rank means more demand. Captures real student behaviour.

Default 30%
F·02

NIRF Rank (NR)

NR = 1 − (nirfRank ÷ 300). NIRF Engineering 2026 ranking, normalised so #1 = 1.00 and #300 = 0.00. Unranked institutions get a 0.30 floor.

Default 20%
F·03

Placement & CTC (PL)

PL = avgCTC ÷ 25 LPA. Outcome-based — average placement CTC normalised to ₹25 L = 1.0. Programs below the family's minimum CTC get a −0.30 penalty.

Default 20%
F·04

Branch Demand (BD)

BD = branchPref ÷ 5. Captures CSE > ECE > Mech > Civil hierarchy. Counsellor edits per-branch chips (1–5) so each family's target branch carries the right weight.

Default 15%
F·05

Location / Exposure (LS)

LS = metroTier(city). Tier-1 metros (Delhi, Mumbai, Bengaluru…) = 1.00 · Tier-2 = 0.70 · Tier-3 = 0.45 · remote = 0.25. Drives internship & exposure access.

Default 10%
F·06

Institute Legacy (IL)

IL = foundingHeritage(inst). Pre-1970 RECs (Trichy, Warangal, Surathkal…) = 1.00 · 2000s NITs = 0.65 · 2009-10 wave NITs = 0.30. Brand age × alumni density.

Default 5%
Bias correction layer · The α term

Some colleges are consistently chosen over what NIRF predicts.

A small additive correction lifts colleges where students vote with their feet — the "hidden gems" effect. Default α = 0.15. Set to 0 to disable; up to 0.30 to amplify.

base     = Σ (s_i × w_i) ÷ Σ w_i        // weighted average, ∈ [0, 1]
adjusted = base + α × (PC − NR)           // bias correction
composite = adjusted × 100               // displayed score

When PC > NR (a college with strong people-choice but lower NIRF rank), the term is positive — the program is boosted. Examples surfaced by α = 0.15: NIT Surathkal CSE branches, NIT Warangal ECE, NIT Trichy Production. Set α = 0 to fall back to a pure weighted average.

The output is a four-bucket choice list — Safe, Target, Reach and Dream — based on closing-rank fit relative to the student's CRL. Sorted by composite score within each bucket, with a NIRF-only reference column always visible. Full mathematical breakdown in the documentation section below.

Documentation · Scoring methodology

How every program gets its composite score.

Six factors. Each normalised to [0, 1]. Weighted, summed, then nudged by a small bias-correction term that surfaces colleges undervalued by NIRF. The full maths in plain language — every formula, every default, every override.

The composite formula

         6
base   = Σ ( s_i × w_i ) ÷ Σw_i      // weighted average
        i=1

adjusted = base + α × (PC − NR)       // bias correction

composite = adjusted × 100             // displayed score

where:
  s_i ∈ [0, 1]      // per-factor sub-score
  w_i ∈ [0, 50]     // weight set by counsellor
  Σ w_i = 100       // weights normalised to 100%
  α     ∈ [0, 0.30] // default 0.15
  PC, NR ∈ [0, 1]   // people-choice & NIRF sub-scores

Programs with missing data (e.g. unranked institutes) get a sensible floor — never a zero — so they can't be unfairly penalised. Recompute time across all 324 programs is < 60 ms.

Sub-score formulas

PC · People Choice — 1 − (closing ÷ maxClosing) with trend correction ×(1 + Δ). Lower closing rank = higher demand. Δ is the year-over-year demand shift; clamped to [−0.25, +0.25].

NR · NIRF Rank — 1 − (nirfRank ÷ 300). Capped at 1.0; unranked institutes floored at 0.30.

PL · Placement — avgCTC ÷ 25 LPA; soft −0.30 penalty if below the family's minimum acceptable CTC.

BD · Branch Demand — branchPref ÷ 5. Defaults: CSE/AI/IT = 5, ECE/VLSI = 4, Mech/EEE = 3, Civil/Chem = 2, Mining = 1. Counsellor adjusts via chips.

LS · Location/Exposure — Tier-1 metro = 1.00 (Delhi, Mumbai, Bengaluru, Chennai, Kolkata, Hyderabad). Tier-2 = 0.70. Tier-3 = 0.45. Remote = 0.25.

IL · Institute Legacy — Pre-1970 RECs (Trichy, Warangal, Surathkal, Rourkela…) = 1.00 · IIEST Shibpur = 1.00 · DTU Delhi = 0.92 · IIIT Hyderabad = 0.80 · 2009-10 wave NITs = 0.30. Captures alumni-network depth and brand age.

Output strategy · Four buckets

Don't generate one list — generate four buckets.

Each program is tagged Safe / Target / Reach / Dream based on the ratio of the student's CRL to the program's 2025 closing rank. The matrix sorts by composite within each bucket, so families always see the strongest options at every risk level.

Special rule · Dream tagging

When the student's CRL is below 100,000, every shortlisted program is auto-tagged 100% Dream-eligible.

Below 1 lakh CRL, even premier NITs and IIITs are within reach. The matrix removes the conservative warning for these students and recomputes the fit pill so the choice list shows the full ambition envelope. Try setting studentRank = 85000 in the matrix above — every row turns Dream.

Guide · Filling your choices

Once allotment opens, every student picks one of three options.

DASA, CIWG and JoSAA all use the same float / freeze / lock decision framework after each round of seat allocation. Picking the right one is as important as the choice list itself — here's how each works and when to use it.

OPTION 01

Float

"Keep what I got, but try for an upgrade in the next round."

You accept the current allotted seat as a backup, but stay in the seat-allocation pool. In the next round, the system will try to upgrade you to a higher-preference choice from your filled list. If it can't, you keep the current seat. Zero risk of losing the current seat — only upside.

Use when: the allotment is acceptable but not your top choice, and you genuinely want a shot at an upgrade.
OPTION 02

Freeze

"This is the one. Lock it in."

You accept the allotted seat as final. You exit the seat-allocation pool and confirm admission. The system will not consider you for any upgrade in subsequent rounds — even if a higher-preference choice would have come through.

Use when: the allotment is your top realistic option, you want certainty, or you're done waiting and want to focus on joining the institute.
OPTION 03

Slide

"Stay in this college, but try for a better branch."

Available only in some allocation systems (notably JoSAA after Round 1). You accept the institute but stay open to a branch upgrade within the same institute. You won't move to a different college, but you might move from, say, Civil to Mechanical at the same NIT.

Use when: you've got into a good college but the branch isn't ideal — and a better branch in the same college is realistically reachable.
Example 01 · The upgrade-seeker

Aarav got NIT Trichy ECE. He wanted NIT Trichy CSE.

The situation: Aarav's CRL is 24,500. After Round 1 of DASA-CIWG allocation, he is allotted NIT Trichy · Electronics & Communication Engineering (his choice #4). His top three filled choices were NIT Trichy CSE, NIT Surathkal CSE, and NIT Calicut CSE — none came through in R1.

DECISION
Float. ECE at Trichy is excellent — he keeps it as his secured seat. But CSE at the same campus is a clear upgrade he genuinely wants. By choosing Float, he stays in the pool: if CSE @ Trichy frees up in R2, he gets it. If not, he keeps ECE @ Trichy. Zero downside.
What he should NOT do: Freeze ECE just because he's tired of waiting. Float costs him nothing and gives a real upgrade chance.
Example 02 · The done-and-dusted

Priya got NIT Surathkal CSE. She is thrilled.

The situation: Priya's CRL is 18,200. After Round 1, she's allotted NIT Surathkal · Computer Science & Engineering — her choice #2. Her #1 was NIT Trichy CSE, which is unlikely to free up given how the cutoffs are trending.

DECISION
Freeze. NIT Surathkal CSE is a top-tier outcome — on par with #1 in placements and reputation. The marginal probability of upgrading to NIT Trichy CSE in R2 is low, and Floating risks anxiety + delayed paperwork. Freezing gets her admission process moving immediately and lets her focus on hostel, fees, and travel from Dubai.
What she should NOT do: Float just because there's a #1 on her list. The bar for Floating is "is the upgrade meaningful AND realistic?" — here it's neither.

NRI Checklist · Documentation

What every NRI / CIWG student needs before the deadline.

DASA and CIWG verification are paperwork-heavy and time-sensitive. Missing one document can cost an allotment. Print this list, gather everything, and have it scanned (PDF, <2 MB each) before counselling opens.

Identity & Citizenship

  • Student's valid passport (front, back, all visa pages — minimum 6 months validity beyond admission date)
  • Aadhaar card (if Indian citizen residing abroad)
  • PAN card of student or sponsor
  • OCI / PIO card for Person of Indian Origin students (if applicable)
  • Birth certificate (notarised English translation if originally in another language)
  • Recent passport-size photographs (10 copies, light background, written name & date on reverse)

Academic Records

  • Class X marksheet & passing certificate (CBSE / ICSE / state board / IGCSE / IB / American)
  • Class XII marksheet & passing certificate
  • JEE Main scorecard (DASA-CIWG) or SAT scorecard (DASA Non-CIWG / SAT-pathway)
  • Equivalence certificate from AIU (Association of Indian Universities) for foreign-board (IB / American / British) Class XII
  • Class XII migration certificate (mandatory for admission, can be applied for after allotment)
  • School conduct / character certificate from the principal of last attended school

Sponsor & NRI Status Proof

  • Sponsor's NRI status proof — valid work visa, residence permit, or PR card (covering at least 1 year prior to application)
  • Sponsor's passport copy (all relevant pages)
  • Sponsor declaration form (provided by DASA, signed and notarised)
  • Sponsor's employment certificate or business licence from the country of residence
  • Sponsor's income proof (last 3 months salary slips or last 1 year tax return)
  • Bank statements (last 6 months) showing sufficient funds for tuition + living
  • Relationship proof between sponsor and student (if sponsor is not a parent — affidavit or notarised letter)

Financial & Admission

  • Demand draft / wire transfer receipt for first-instalment fees in USD (per institute's published tuition)
  • Educational loan sanction letter (if applicable — from Indian or NRE bank)
  • Signed anti-ragging affidavit (student) per UGC format
  • Signed anti-ragging affidavit (parent/guardian) per UGC format
  • Medical fitness certificate from a registered physician (some institutes also require an HIV / Hep B test)

Optional but Recommended

  • Notarised English translations of any non-English documents
  • Apostille / attestation from the country of residence's MEA (where required)
  • Photocopy set of all of the above (some institutes keep originals on file at admission)

Three things that trip up most NRI families

  1. AIU equivalence takes 2–3 weeks. If your child is on IB or American board, start this before JEE results — the certificate is non-negotiable.
  2. Sponsor income proof must be current. "Last 3 months" means within 90 days of submission — not last year's payslip.
  3. Demand drafts to Indian banks from abroad take 5–10 working days. Wire transfers are faster but the receipt format differs by bank — get it confirmed in advance.
