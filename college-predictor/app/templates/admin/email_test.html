Reading the result

  • Email arrivesAll automated emails will work. Done.
  • Resend API 403
    “domain is not verified”     Add the DNS records Resend gives you for eduaakashaa.in and wait for it to show Verified. Until then Resend only allows sending to your own account address.
  • Resend API 401 / 422 401 = bad RESEND_API_KEY. 422 = the MAIL_FROM address isn't on a verified domain.
  • Network is unreachable
    (transport = smtp)     Render blocks outbound SMTP. Set RESEND_API_KEY so the app sends over HTTPS instead.
  • No transport configured Neither RESEND_API_KEY nor the SMTP_* vars are reaching the app — check the Environment tab and redeploy.
