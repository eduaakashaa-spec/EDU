Reading the result

  • Email arrivesAll automated emails will work. Done.
  • SMTPAuthenticationError
    “Username and Password not accepted”     SMTP_PASS must be a Google App Password (16 chars), not your account password. Google Account → Security → 2-Step Verification → App passwords.
  • Timeout / Connection refused Outbound SMTP is blocked by the host (common on free tiers). Fix by switching to an HTTP email API (Resend / SendGrid / Brevo) instead of SMTP.
  • SMTP not configured The SMTP_* environment variables aren't reaching the app — check the Environment tab and redeploy.
